# Anonymous remailer
---
## Introduzione
> Un **anonymous remailer** e' un server che, ricevuta una [[posta-elettronica|email]], _la inoltra al destinatario rimuovendo le informazioni del mittente_.

E' fondamentalmente un [[proxy|proxy]] per le email.
